# SlidePane 右侧面板

## 何时使用

从页面右侧弹出面板，展示相应内容。用户在抽屉内操作时不必离开当前任务，操作完成后，可以平滑地回到原任务。
常用于： 查看详情， 预览较多内容的场景下。

## API

| 参数      | 说明                   | 类型            | 默认值 |
| --------- | ---------------------- | --------------- | ------ |
| visible   | SlidePanel 是否可见    | `boolean`       | -      |
| className | 右侧面板外层容器的类名 | `string`        | -      |
| style     | 右侧面板外层容器的样式 | `CSSProperties` | -      |
| bodyStyle | 右侧面板内部容器的样式 | `CSSProperties` | -      |
| onClose   | 关闭回调               | `function`      | -      |
