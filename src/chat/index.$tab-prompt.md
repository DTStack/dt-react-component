## API

| 参数        | 说明                | 类型              | 默认值 |
| ----------- | ------------------- | ----------------- | ------ |
| content     | 文案                | `string`          | -      |
| className   | 类名                | `string`          | -      |
| extraRender | 额外的 dom 结构渲染 | `React.ReactNode` | -      |
