# TinyTag 小标签

## 何时使用

TinyTag 组件通常用于在某些文案后面，用于标识当前行数据的分类

## API

### TinyTag

| 参数      | 说明       | 类型              | 默认值 |
| --------- | ---------- | ----------------- | ------ |
| value     | 标签文案   | `string`          |        |
| icon      | 自定义图标 | `React.ReactNode` |        |
| className | class 名称 | `string`          |        |
