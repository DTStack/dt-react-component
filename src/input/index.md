## API

### 类型

| 参数               | 说明                   | 类型       | 默认值 |
| ------------------ | ---------------------- | ---------- | ------ |
| onPressEnter       | 输入英文回车的回调函数 | `Function` |
| onPressEnterNative | 输入任意回车的回调函数 | `Function` |        |

其余属性均继承自 `Input` 组件，参考 [Input API](https://4x.ant.design/components/input-cn/#Input)
