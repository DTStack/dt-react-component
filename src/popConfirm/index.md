## API

| 属性     | 说明         | 类型                               | 默认值    |
| -------- | ------------ | ---------------------------------- | --------- |
| showIcon | 是否显示图标 | boolean                            | true      |
| type     | 图标类型     | 'primary' \| 'warning' \| 'danger' | 'primary' |

其余属性均继承自 `Popconfirm` 组件，参考 [Popconfirm API](https://4x.ant.design/components/popconfirm-cn/#API)
