## API

### Returns

| 参数       | 说明          | 类型                   | 默认值 |
| ---------- | ------------- | ---------------------- | ------ |
| ref        | DOM 实例      | `HTMLDivElement`       | -      |
| rect       | DOMRect       | `UseMeasureRect`       | -      |
| getElement | 获取 DOM 实例 | `() => HTMLDivElement` | -      |
